# 🎯 高级数据可视化网站

一个功能完整的专业数据可视化平台，基于Chart.js构建，支持动态图表切换、响应式设计和丰富的交互功能。

## 📊 项目概述

本项目是一个单页式数据可视化仪表板，精准展示6大主题模块的业务数据，采用现代化设计和商务科技风格，提供企业级的数据可视化体验。

### 🎨 主要特色

- ✨ **动态图表切换** - 每个图表支持3种以上可视化类型
- 🎯 **智能导航系统** - 快速定位和自动高亮
- 📱 **完全响应式** - 适配PC、平板、手机等多种设备
- 🌓 **主题切换** - 支持明暗两种主题模式
- 🔄 **实时交互** - 平滑动画和丰富的用户反馈

## 🎮 功能说明

### 图表交互
- **切换图表类型**: 点击图表控制区域的切换按钮
- **全屏查看**: 点击全屏图标进入独立展示模式
- **数据悬停**: 鼠标悬停查看详细数值信息
- **平滑滚动**: 点击导航项目平滑定位到对应区域

### 主题切换
- **明暗主题**: 点击顶部导航栏的主题切换按钮
- **本地记忆**: 主题选择会自动保存到浏览器本地存储
- **快捷键**: `Ctrl + D` 快速切换主题

### 导航功能
- **顶部导航**: 固定在页面顶部，支持快速跳转
- **侧边栏导航**: 可折叠式设计，移动端友好
- **自动高亮**: 滚动时自动高亮当前浏览区域
- **返回顶部**: 右下角浮动按钮，智能显示/隐藏

### 响应式适配
- **桌面端** (1920px+): 双栏布局，完整功能展示
- **平板端** (768px+): 自适应网格布局，触摸优化
- **手机端** (<768px): 单栏布局，汉堡菜单

## 🎨 定制指南

### 修改数据
编辑 `advanced-script.js` 中的数据对象：

```javascript
// 智能手表数据示例
const smartWatchData = {
  quarters: ['Q1', 'Q2', 'Q3', 'Q4'],
  sales: [120, 150, 130, 180]  // 修改此处数据
};
```

### 添加新图表
1. 在HTML中添加新的图表容器
2. 在CSS中添加对应样式
3. 在JavaScript中创建新的图表实例

### 自定义配色
修改 `advanced-styles.css` 中的CSS变量：

```css
:root {
  --primary-color: #3b82f6;    /* 主色调 */
  --secondary-color: #8b5cf6;  /* 辅助色 */
  --gradient-start: #3b82f6;   /* 渐变起始色 */
  --gradient-end: #8b5cf6;     /* 渐变结束色 */
}
```
